жіЁж„ҸпјҒ

жң¬ж•ҷзЁӢдёӯдҫқиө–еҺҹдҪңиҖ…mritdзҡ„д»“еә“зҡ„йғЁеҲҶпјҲеҰӮDockerе®№еҷЁйғЁзҪІпјүпјҢз”ұдәҺеҲ еә“и·‘и·Ҝзҡ„еҺҹеӣ жІЎжңүеӨҮд»ҪпјҢжҒўеӨҚзҡ„е·ҘдҪңйҮҸжҜ”иҫғеӨ§пјҢзӣ®еүҚж— жі•дҪҝз”ЁпјҒ

дёҖгҖҒTPClash дҪҝз”Ё

1.1гҖҒзӣҙжҺҘеҗҜеҠЁ

TPClash еҸӘжңүдёҖдёӘдәҢиҝӣеҲ¶ж–Үд»¶, зӣҙжҺҘд»Һ Release йЎөйқўдёӢиҪҪдәҢиҝӣеҲ¶ж–Үд»¶иҝҗиЎҢеҚіеҸҜ. TPClash дәҢиҝӣеҲ¶еҶ…еөҢе…ҘдәҶзӣ®ж Үе№іеҸ°зҡ„ Clash дәҢиҝӣеҲ¶ж–Үд»¶д»ҘеҸҠе…¶д»–иө„жәҗж–Үд»¶(All in one), еҗҜеҠЁеҗҺдјҡиҮӘеҠЁйҮҠж”ҫ, жүҖд»Ҙж— йңҖеҶҚдёӢиҪҪ Clash.

жіЁж„Ҹ: TPClash й»ҳи®ӨдјҡиҜ»еҸ–дҪҚдәҺ /etc/clash.yaml зҡ„ clash й…ҚзҪ®ж–Үд»¶, еҰӮжһң clash й…ҚзҪ®ж–Үд»¶еңЁе…¶д»–дҪҚзҪ®иҜ·иҮӘиЎҢдҝ®ж”№.

1.2гҖҒSystemd е®үиЈ…

йҷӨдәҶзӣҙжҺҘиҝҗиЎҢд№ӢеӨ–, й’ҲеҜ№дәҺж”ҜжҢҒ Systemd зҡ„зі»з»ҹ TPClash д№ҹж”ҜжҢҒ install е‘Ҫд»Өз”ЁдәҺе°ҶиҮӘиә«е®үиЈ…дёә Systemd жңҚеҠЎ; е®үиЈ…ж—¶ TPClash е…Ҳе°ҶиҮӘиә«еӨҚеҲ¶ еҲ° /usr/local/bin/tpclash, з„¶еҗҺеҲӣе»ә /etc/systemd/system/tpclash.service й…ҚзҪ®ж–Үд»¶, е№¶дё”е°Ҷйҷ„еҠ еҸӮж•°д№ҹеҗҢжӯҘеҶҷе…ҘеҲ° Systemd й…ҚзҪ®дёӯ.

1.3гҖҒDocker иҝҗиЎҢ

жіЁж„Ҹ: д»Һ v0.1.0 зүҲжң¬ејҖе§Ӣ, еҰӮжһңдҪҝз”Ё Docker иҝҗиЎҢжҲ–иҖ…е®ҝдё»жңәе®үиЈ…дәҶ Docker, TPClash дјҡиҮӘеҠЁе°қиҜ•дҪҝз”Ё nftables иҝӣиЎҢдҝ®еӨҚ; еҰӮжһңе®ҝдё»жңәдёҚж”ҜжҢҒ nftables, иҜ·иҮӘиЎҢдҪҝз”Ё iptables -I DOCKER-USER -i src_if -o dst_if -j ACCEPT е‘Ҫд»Өдҝ®еӨҚ.

еҰӮжһңжғіиҰҒеңЁ Docker дёӯдҪҝз”Ё tpclash, еҸӘйңҖиҰҒжҢӮиҪҪеӨ–йғЁй…ҚзҪ®ж–Үд»¶еҚіеҸҜ:

жӯӨе‘Ҫд»ӨеҒҮи®ҫй…ҚзҪ®ж–Үд»¶дҪҚдәҺе®ҝдё»жңәзҡ„ /root/clash.yaml дҪҚзҪ®, е…¶д»–дҪҚзҪ®иҜ·иҮӘиЎҢжӣҝжҚў; иҜҘй•ңеғҸйҮҮз”Ё Earthly зј–иҜ‘, Earthfile еӯҳеӮЁеңЁ autobuild д»“еә“.

1.4гҖҒе®№еҷЁеҢ–иҷҡжӢҹжңәйғЁзҪІ

TPClash дёҖејҖе§Ӣзҡ„зӣ®ж Үе°ұжҳҜдҪңдёәдёҖдёӘзЁіе®ҡеҸҜйқ зҡ„гҖҒеҸҜд»ҘзӣҙжҺҘжүҳз®Ўй…ҚзҪ®зҡ„еҶ…зҪ‘зҪ‘е…ідҪҝз”Ё, иҷҪз„¶ TPClash еҸҜд»Ҙе…је®№еӨ§еӨҡж•°зі»з»ҹ, дҪҶзү№ж®Ҡзі»з»ҹзҺҜеўғдҫӢеҰӮ OpenWrt зӯүеҸҜиғҪдјҡеҮәзҺ° дёҖдәӣе…је®№жҖ§й—®йўҳ;дёәдәҶз»ҹдёҖйғЁзҪІзҺҜеўғе’Ңжӣҙж–№дҫҝдҪҝз”Ё, зӣ®еүҚе·ІеўһеҠ дәҶзәҜе®№еҷЁеҢ–зі»з»ҹ Flatcar зі»з»ҹж”ҜжҢҒ, иҜҘзі»з»ҹж”ҜжҢҒзӣҙжҺҘдҪҝз”ЁеҚ•дёӘй…ҚзҪ®ж–Үд»¶е®ҢжҲҗеј•еҜје’ҢиҮӘеҠЁеҢ–йғЁзҪІ, дё”еҗҺеҸ°иҮӘеҠЁ ж»ҡеҠЁеҚҮзә§; еҸҜд»ҘжҢҒз»ӯз»ҙжҢҒзі»з»ҹзҡ„жңҖж–°зҠ¶жҖҒе№¶дё”еҸҜе®үе…Ёеӣһж»ҡ; д»ҘдёӢжҳҜеңЁ ESXi дёӯзӣҙжҺҘйғЁзҪІиҜҙжҳҺ:

  • 1гҖҒдёӢиҪҪйЎ№зӣ®еҶ…зҡ„ flatcar.butane.yaml
  • 2гҖҒи°ғж•ҙй…ҚзҪ®ж–Үд»¶еҶ…зҡ„ IP ең°еқҖе’ҢзҪ‘е…іең°еқҖ, DNS дёҖиҲ¬дёҚйңҖиҰҒдҝ®ж”№
  • 3гҖҒи°ғж•ҙй…ҚзҪ®ж–Үд»¶еҶ… TPClash зҡ„еҗҜеҠЁе‘Ҫд»Ө, дёҖиҲ¬йңҖиҰҒжҢҮе®ҡиҝңзЁӢ clash й…ҚзҪ®ж–Үд»¶ең°еқҖ
  • 4гҖҒеҸӮиҖғ butane е®ҳж–№ж–ҮжЎЈе®үиЈ… butane е·Ҙе…·
  • 5гҖҒжү§иЎҢ butane --pretty --strict flatcar.butane.yaml | base64 -w0 з”ҹжҲҗ base64 зј–з Ғж јејҸзҡ„ Ignition й…ҚзҪ®(yamlеҗҚз§°иҮӘиЎҢжӣҝжҚў)
  • 6гҖҒдёӢиҪҪеҜ№еә”зі»з»ҹзҡ„ ova зі»з»ҹйғЁзҪІж–Үд»¶
  • 7гҖҒеңЁ ESXi еҶ…еҲӣе»әиҷҡжӢҹжңәйҖүжӢ©д»Һ OVA йғЁзҪІ
  • 8гҖҒйғЁзҪІж—¶ е…¶д»–и®ҫзҪ®/Options дёӯ Ignition/coreos-cloudinit data еЎ«еҶҷз¬¬ 5 жӯҘз”ҹжҲҗзҡ„ base64 еӯ—з¬ҰдёІ, Ignition/coreos-cloudinit data encoding еЎ«еҶҷ base64
  • 9гҖҒжңҖеҗҺеҗҜеҠЁе®ҢжҲҗ, дёҖдёӘе®№еҷЁеҢ–гҖҒдёҚеҸҜеҸҳзҡ„еҸҜйқ  TPClash зҪ‘е…іе°ұеҗҜеҠЁдәҶ

е…ідәҺиҝҷдёӘзі»з»ҹд»ҘеҸҠе…¶й…ҚзҪ®ж–Үд»¶йҷҗдәҺзҜҮе№…ж— жі•еҒҡиҝҮеӨҡиҜҙжҳҺ, жҺЁиҚҗйҳ…иҜ»еҚҡе®ўж–Үз« .

1.5гҖҒи®ҫзҪ®жөҒйҮҸиҪ¬еҸ‘

TPClash еҗҜеҠЁжҲҗеҠҹеҗҺ, е°Ҷе…¶д»–дё»жңәзҡ„зҪ‘е…іжҢҮеҗ‘еҪ“еүҚ TPClash жңҚеҠЎеҷЁ IP еҚіеҸҜе®һзҺ°йҖҸжҳҺд»ЈзҗҶ; еҜ№дәҺиў«д»ЈзҗҶдё»жңәиҜ·дҪҝз”Ёе…¬зҪ‘ DNS.

иҜ·дёҚиҰҒе°Ҷе…¶д»–дё»жңәзҡ„ DNS д№ҹи®ҫзҪ®дёә TPClash жңҚеҠЎеҷЁ IP, еӣ дёәиҝҷеҸҜиғҪеҜјиҮҙдёҖдәӣдёҚеҸҜйў„жөӢзҡ„й—®йўҳ, е…·дҪ“иҜ·еҸӮиҖғ Clash DNS з§‘жҷ®.

1.6гҖҒеҚҮзә§ TPClash

еҜ№дәҺдәҢиҝӣеҲ¶ж–Үд»¶йғЁзҪІзҡ„з”ЁжҲ·, еҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»ӨеҚҮзә§еҲ°жңҖж–°зүҲжң¬:

еҚҮзә§еүҚиҜ·зЎ®дҝқе…ій—ӯдәҶ tpclash жңҚеҠЎ, еҚҮзә§ж—¶й»ҳи®ӨдҪҝз”Ё https://mirror.ghproxy.com иҝӣиЎҢеҠ йҖҹ, еҰӮжһңдёҚжғідҪҝз”ЁеҸҜд»ҘйҖҡиҝҮ --with-ghproxy=false йҖүйЎ№е…ій—ӯ.

1.7гҖҒзҪ‘йЎөжҺ§еҲ¶йқўжқҝ

TPClash иҮӘеёҰдёҺеҶ…ж ёзӣёйҖӮй…Қзҡ„зҪ‘йЎөжҺ§еҲ¶йқўжқҝпјҢжӮЁеҸӘйңҖиҰҒи®ҝй—® IP:9090/ui еҗҺпјҢеҶҚе°Ҷ Clash еҗҺз«Ҝең°еқҖеЎ«е…Ҙ пјҲй»ҳи®ӨжҸҗдҫӣзҡ„ example.yaml и®ҫзҪ®жЁЎжқҝзҡ„ RESTful API ең°еқҖжҳҜ IP:9090 пјү еҚіеҸҜи®ҝй—®гҖӮ

2.3гҖҒMeta з”ЁжҲ·

жіЁж„Ҹ: Meta зүҲжң¬жҡӮж—¶жІЎжңүз»ҸиҝҮдёҘж јзҡ„жөӢиҜ•, дҪңиҖ…е№¶жІЎжңүдҪҝз”Ё Meta зүҲжң¬зҡ„йңҖжұӮ.
