薅秃大厂羊毛！10 分钟利用 Cloudflare Pages 零成本搭建永久白嫖节点指南

发表于2026-05-23|更新于2026-07-23|proxyedgetunnel
|阅读量:

在网络折腾圈里，“长期、稳定、免费”这三个词很少能同时出现。但利用 Cloudflare 的全球边缘网络，我们确实可以搞出一套不需要买服务器、不需要掏域名费、无限复活且极度稳健的科学上网轻量化替代方案。

今天这篇博文就带大家空手套白狼，用 Cloudflare Pages + 免费次级域名，10 分钟在本地跑起一套属于你自己的 VLESS / Trojan 专属路由。

🛠️ 第一步：空手套白狼，白嫖永久免费域名

工欲善其事，先搞个域名。我们不需要去万网或 Namecheap 掏钱，直接去白嫖提供商那里注册。

  1. 注册平台： 登录主流的免费域名分发平台。

    第1种：【点击前往】

    第2种：【点击前往】

  2. 白嫖额度： 账户登录后，可以直接免费申请多达 5 个永久域名。

  3. 后缀选择： 强烈建议根域名直接选择 .ccwu.cc 或 .us.ci，这两个后缀在 Cloudflare 上的兼容性非常好，且完全免费。

☁️ 第二步：Cloudflare 后台基础设施配置

拿到域名后，直接登录你的 Cloudflare (CF) 【点击注册】控制台，将刚刚申请的免费域名添加并托管进去。接下来的核心都在 CF 的边缘计算面板里操作。

1. 创建 KV 空间（存储节点配置）

进入 CF 菜单：存储和数据库 (Storage & Databases) -> Workers KV。

  • 点击创建命名空间，命名为 KV（或者你喜欢的任意名字），作为后面的后端对接。

2. 创建 Pages 项目并部署源码

进入 CF 菜单：**Workers 和 Pages (Workers & Pages)**。

  • 切换到 Pages 选项卡，选择手动上传。

  • 上传由社区大牛 CMliu 开源的专属部署包。

    【**点击下载**】

    【重要提醒！】：大陆用户切勿使用！否则有法律风险，请遵守当地法律法规，限海外用户使用！

🚀 第三步：完善环境变量与网关绑定

代码上传成功后别急着走，还需要补全两项关键的安全与自定义配置。

1. 配置安全密码（环境变量）

  1. 在 Pages 项目控制台，进入 设置 (Settings) -> **环境变量 (Environment Variables)**。
  2. 为生产环境定义变量：
    • 变量名称： ADMIN
    • 变量值： 填入你的自定义管理员密码（防止别人爆破你的节点面板）。
  3. 保持保存，返回 部署 (Deployment) 选项卡，点击“创建新部署”，重新上传一遍安装包让变量生效。

2. 绑定 KV 命名空间

  1. 在 设置 -> 绑定 (Functions) 下，点击添加 KV 命名空间。
  2. 变量名称 严格填写：KV
  3. 选择你第一步创建的那个 KV 空间，保存。

3. 映射 CNAME 自定义次级域名

由于 Cloudflare 默认分配的 .pages.dev 域名在国内基本处于被墙或极端阻断状态，必须绑定你自己的自定义域名才能正常使用。

  1. 进入 Pages 项目的 自定义域 (Custom domains) 选项卡，点击设置自定义域。
  2. 注意： 不要用你的免费根域名，填入一个自定义的次级域名，例如 abc.ccwu.cc。
  3. 按照提示，去你的域名 DNS 解析里，添加一条 CNAME 记录，指向 edgetunnel.pages.dev，等待激活成功。

📊 第四步：登录面板，获取你的订阅节点

大功告成！现在，打开浏览器直接访问你的专属地址：

输入你在环境变量里设置的 ADMIN 密码，就能直接进入内置的可视化极客管理后台。

🛠️ 核心玩法与高级配置

这个后台非常强大，它原生支持 VLESS 和 Trojan 协议，并深度集成了加密传输。

  • 新手无脑流： 直接复制后台生成的节点链接或自适应订阅地址，一键导入到你手机或电脑的客户端（如 V2RayN、Clash 或 Sing-box）。

  • 极客进阶流： 如果你对网络延迟有极致追求，可以在后台顶部点击 “我是高手！我就是要折折腾！”，在里面填入优选的 ProxyIP（如 ProxyIP.US.CMLiussss.Net 等海内外优质边缘 IP），配合优选 API，让白嫖节点的测速也能轻松秒开 4K 甚至 8K 视频。

📝 总结

通过 Cloudflare Pages 部署节点的爽点在于，只要 CF 不倒闭，你的节点就永远拥有“无限复活”的底层肉盾。零服务器成本、数据全在自己的 KV 空间，用来作为主力梯子的备用线或者轻量生产力工具，性价比直接拉满。
