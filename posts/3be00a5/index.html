Debian 13 安装教程

发表于2025-09-03|更新于2025-11-01|osubuntudebian
|阅读量:

你是不是也听过很多人说：Linux 很强大，但安装起来特别麻烦？
其实真相是 —— 如果有人带着你走一遍，安装 Debian 13 甚至比装个 Windows 还轻松。
今天我就带你从 0 开始，顺顺利利把系统搞定。

📥 第一步：下载 Debian 13 镜像

1.打开 Debian 官方下载页面https://www.debian.org/

2.建议下载 netinst（网络安装版），体积小，安装时会联网获取最新软件包镜像文件后缀一般是 .iso

💡 小技巧：在国内推荐使用 清华大学 / 中科大开源镜像站，速度更快。

💾 第二步：制作启动盘

准备一个 8GB 以上 U 盘，然后使用工具写入镜像：

  • Windows 用户推荐：Rufus https://rufus.ie/zh/

  • macOS / Linux 用户推荐：balenaEtcher https://etcher.balena.io/

⚠️ 注意：写入镜像会清空 U 盘数据，请提前备份！

⚙️ 第三步：设置 BIOS/UEFI

  1. 插上 U 盘，重启电脑
  2. 开机时按 Del / F2 / F12 进入 BIOS
  3. 设置 USB 启动优先级

🖥️ 第四步：开始安装 Debian

1.选择 Graphical Install（图形化安装）

2.按提示选择 语言（中文）、位置（中国）、配置键盘（汉语）

3.配置网络（建议用有线网络，稳定可靠）

（1）主机名相当于你服务器存在网络的名称，随便起一个，自己知道就行

（2）由于我们是本地搭建服务器，所以这里可以不填，删除即可

4.设置用户名和密码

（1）首先需要给root账户设置密码

（2）创建一个普通用户，输入用户名

（3）创建普通账户密码

5.分区：

    • 新手推荐 Guided - use entire disk（自动分区）
    • 熟手可以手动分区

下面以新手推荐方式进行分成操作。熟手请随意。

（1）选择使用整个磁盘

（2）需要要进行磁盘分区的磁盘

（3）使用新手推荐方式，将所有文件放在同一个分区中

（4）选择“完成分区操作并将修改写入磁盘”

（5）确定进行写入操作

点击继续后，会进行安装包解压操作。稍等片刻

如果提示“扫描额外的安装介质” 选择 “否”即可。

6.配置软件包管理器

（1）如果你在国内，选择中国即可。

（2）这里需要选择一个镜像站点。目前我这边比较快的是：mirrors.tuna.tsinghua.edu.cn 也就是清华大学镜像站点。根据你目前所在的位置，可以使用ping 命令测试一下 这几个站点哪个一个比较好，就用哪一个即可。

（3）如果你是本地搭建服务，这里留空即可。

之后将进入漫长的安装操作。

如果中途提示“您要参加软件包流行度调查吗？” 直接选 “否”即可。

7.选择软件：

    • 新手推荐：

      ✅ GNOME 桌面环境

      ✅ 标准系统工具

      ✅ SSH server

    • 想轻量一点或你电脑配置比较低，桌面环境就选xfce或LXDE：

      ✅ XFCE/LXDE桌面环境

      ✅ 标准系统工具

      ✅ SSH server

等待安装完成 → 拔掉 U 盘 → 系统重启 → 进入 Debian 13！

🚀 第五步：安装完成后的第一步

输入用户名和密码，进入桌面。

👉 这样可以保证系统更新到最新状态。

🎉 总结

到这里，你就完成了 Debian 13 的安装！
