RSS阅读器与知识管理利器 Glean 拾灵

发表于2025-12-05|更新于2025-12-09|toolsdocker
|阅读量:

简介

什么是 Glean ？

Glean（拾灵）是一款开源、可自托管的 RSS 阅读器和个人知识管理工具，旨在帮助重度信息消费者通过智能 RSS 聚合与管理功能，解决信息过载问题，帮助用户高效地进行阅读、筛选和知识沉淀。

主要特点

  • 📰 RSS 订阅：轻松订阅和管理 RSS/Atom 源，支持 OPML 导入导出
  • 📚 智能阅读：简洁、无干扰的阅读体验，支持内容过滤
  • 🔖 稍后阅读：保存文章以便稍后阅读，并支持自动清理
  • 📁 文件夹与标签：通过多层级文件夹和标签系统地组织内容
  • ⭐ 收藏系统：收藏重要的订阅文章或外部链接
  • 🔧 后台同步：每 15 分钟自动在后台更新订阅源，确保内容常新
  • 🔒 自托管：通过 Docker 部署，数据完全由您掌控
  • 🎨 现代界面：美观且响应式的用户界面，提供舒适的暖色深色主题
  • 👨‍💼 管理后台：提供用户管理和系统监控功能

对于每天需要处理大量信息源的用户来说，Glean 提供了一个统一的平台来聚合、筛选和消费内容。无论您是研究人员、开发者、学生还是内容创作者，Glean 都能帮助您构建一个个性化的信息流，将分散的知识整理成有序的个人知识库。

安装

在群晖上以 Docker 方式安装。

官方提供了两种部署模式：

  • 一种是精简版部署，仅包含核心的 Web 应用，不带管理后台，适合个人使用；
  • 另一种是完整版部署，包含一个管理后台，提供用户管理和系统统计功能；

本文采用完整版部署 ，由多个服务组成，采用 docker-compose 进行安装，这也是在群晖上部署多容器应用最便捷的方式。

建目录

通过 SSH 客户端连接到您的群晖，然后执行以下命令，创建一个用于存放 Glean 数据的目录

env.txt

env.txt 是环境变量文件

下面是参数的简单说明，请根据需要进行修改

环境变量参考值说明
POSTGRES_DBgleanPostgreSQL 数据库名称。
POSTGRES_USERgleanPostgreSQL 数据库用户。
POSTGRES_PASSWORDgleanPostgreSQL 数据库密码。
SECRET_KEYJWT 签名密钥，建议在生产环境中更改。可使用 openssl rand -hex 32 生成新的密钥。
DEBUGfalse是否启用调试模式，开启后可以在 /api/docs 访问 API 文档。
WEB_PORT8530Web 界面使用的端口号。
ADMIN_PORT8531管理仪表板使用的端口号（仅在完整部署时使用）。
CREATE_ADMINtrue首次启动时是否自动创建管理员账户，设置为 true 将创建默认管理员账户。
ADMIN_USERNAMEadmin默认管理员用户名。
ADMIN_PASSWORDlaosu123默认管理员密码。
ADMIN_ROLEsuper_admin默认管理员角色。
DATABASE_URL数据库的完整 URL，如果没有设置则自动生成。
REDIS_URLredis://localhost:6379/0Redis 服务的 URL。
CORS_ORIGINS["http://localhost", "http://localhost:3000"]允许的跨源请求（CORS）地址的 JSON 数组。
