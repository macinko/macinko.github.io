Windows 11 彻底卸载与清理系统自带 AI（Copilot / Recall）指南

发表于2026-05-25|更新于2026-07-02|oswindows
|阅读量:

随着 Windows 11 的版本迭代（如 25H2 及后续版本），微软在系统底层、任务栏、Edge 浏览器、甚至画图和记事本等传统组件中，强制集成了大量的 AI 功能与高级数据采集服务。这不仅导致系统体量日益臃肿，也引发了部分用户对隐私和系统资源的担忧。

本文将为您介绍如何利用开源工具与一键脚本，彻底移除、禁用这些无法通过常规渠道卸载的系统级 AI 组件，还您一个纯净、清爽的操作系统。

🛠️ 第一步：使用 BCUninstaller 批量卸载常规应用

对于系统内可以通过常规或半隐藏渠道卸载的第三方应用及部分微软自带组件，推荐使用开源免费的 Bulk Crap Uninstaller（简称 BCU）。

  • 功能特点：支持自动化批量卸载，能够深度扫描并清理注册表残留、垃圾文件及隐藏的安装包。
  • 下载地址：BCuninstaller 官方下载 ｜ 备用下载地址

💡 注意：对于被 Windows 系统深度锁定、保护的底层核心 AI 组件（例如饱受争议的 Recall “召唤” 功能），BCU 无法直接将其强制删除。此时需要配合下文的专用脚本进行深度清理。

🚀 第二步：运行 Remove Windows AI 脚本（深度核心清理）

Remove Windows AI 是一款专门用于剥离 Windows 11 系统底层 AI 依赖的开源脚本。它通过 CBS（基于组件的服务）商店级干预，强行解锁并移除那些“不可卸载”的软件包。

核心功能 breakdown：

  • 注册表级彻底禁用：关闭 Copilot（含 Edge 浏览器侧边栏）、Recall 召唤功能、输入洞察以及打字数据采集。
  • 应用组件剥离：移除画图（Paint）中的 AI 图像生成、记事本（Notepad）中的 AI 重写功能，以及系统设置中的 “AI 组件” 页面。
  • 防止二次安装：通过安装自定义 Windows 更新包，锁死 CBS 存储库，防止后续系统更新自动重装 AI 包。
  • 经典应用恢复：脚本提供可选方案，支持用无 AI 的经典版记事本、画图、截图工具及旧版照片查看器替代现代 AI 版本。

使用方法：

  1. 右键点击 Windows 开始徽标，选择 「终端管理员」 或 「PowerShell (管理员)」。
  2. 复制并粘贴以下一键运行命令，按回车执行：
  1. 在弹出的图形化操作面板中，勾选需要清除的 AI 项目（或选择全选），点击 「Apply」 即可。

⚙️ 第三步：使用 Winhance 全方位净化与性能优化

如果您希望在移除 AI 后，进一步精简系统、关停臃肿的后备进程，可以使用 Winhance 这款开源的 Windows 增强实用工具。

主要优化方向：

模块功能说明
软件与应用管理一键卸载或管理 Microsoft Edge、OneDrive、Copilot、Recall 及其他隐藏可选功能；支持新机一键安装常用无广告外部软件。
隐私与安全关闭 Windows 默认的遥测数据上传、隐私追踪和后台广告标识符。
