qnap 威联通 container station docker 安装 homeassistant

发表于2023-07-23|更新于2025-11-01|oswebhomeassistantqnap
|阅读量:

一、在文件管理器上新建HomeAssistant的配置文件夹


在NAS根目录新建共享文件夹 docker
在docker里新建文件夹homeassistant

二、安装HomeAssistant

1.打开App里面的Container Station

2.安装HomeAssistant的镜像


选择homeassistant/home-assistant的镜像，点击创建

3. 配置HomeAssistant

3.1 CPU内存限制，可根据自己的实际资源情况调整


CPU限制选50%，内存选1024M

3.2 环境变量设置

高级设置——环境变量增加 TZ=Asia/Shanghai 即可，其它的会在创建容器后自行增加

3.3 主机模式选择Host模式

3.4 将容器的配置文件夹映射到本地，方便更改配置


共享文件夹——挂载本机共享文件夹——选 /docker/homeassistant
挂载路径 /config

3.5 点击Create，等待容器创建

3.6 安装完毕

打开：威联通ip地址:8123/
就能打开HomeAssistant的管理页面
