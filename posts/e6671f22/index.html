8G 显存也能玩转无审查AI 视频？开源模型 Sulphur 2 本地部署全攻略！

发表于2026-05-23|更新于2026-07-23|toolssoftwareai
|阅读量:

最近 AI 视频生成领域出了一个“不速之客” —— Sulphur 2。它被不少社区网友戏称为“开源白嫖版 Sora”。

和那些条条框框极多的商业模型不同，它最大的卖点只有四个字：真正无审查。因为没有了云端的内容过滤器，各种充满想象力（甚至需要打码）的敏感视频在社区满天飞，讨论度直接拉满。

更关键的是，它不是那种只能躺在顶级服务器里的“吞金兽”。Sulphur 2 主打本地部署，经过社区的魔法优化，如今 8G 显存的消费级显卡也能勉强上车！

不用昂贵的云端订阅，不用排队，没有 API 限制，只要你的硬盘够大、显卡够顶，生产力（或者私家福利）就在你的本地电脑里自由流淌。今天这篇博文就带大家用 ComfyUI 搞定它的本地部署。

🛠️ 部署前置准备

Sulphur 2 的部署主要依赖于大家熟知的 ComfyUI 生态。无论你是硬核 Windows 玩家，还是想在 Mac/PC 上折腾的极客，都可以通过以下步骤完成。

1. 升级/下载 ComfyUI 客户端

由于 Sulphur 2 使用了较新的架构和 GGUF 格式，老旧版本的 ComfyUI 极易报错。

  • 官方客户端下载：【点击前往】或 【备用打包下载】

💡 Tip： 如果你已经安装过旧版，强烈建议直接点击客户端更新，或使用 update.bat 升级到最新版本。

2. 模型下载（根据你的显存量力而行）

Sulphur 2 目前在社区主要分为两个版本，请对号入座：

显存大小推荐模型版本格式与精度说明
32G 以上官方满血版bf16 精度视频生成质量最高，细节最爆炸
8G - 24G官方压缩版 / 蒸馏版fp8 精度 或 GGUF 格式对显存极其友好，生成速度快，性价比首选
  • 官方原版模型下载： Sulphur 2 官方模型【点击前往】或 【网盘下载】、【打包下载】
  • 低显存 GGUF 蒸馏版下载： Sulphur 2 GGUF 模型【点击前往】或 【网盘下载】、【备用下载】

⚠️ 注意： 如果你下载的是 GGUF 格式 的模型，请务必确保你的 ComfyUI 已经安装了 ComfyUI-GGUF 插件，否则无法识别。

🚀 核心玩法：工作流配置

模型下载完成后，将它们放入 ComfyUI 对应的 models/checkpoints 或 models/unet 文件夹中（取决于你使用的是原生格式还是 GGUF 插件路径）。

🖼️ 玩法一：图生视频（Img2Vid）

想要让你珍藏的二次元妹子或者风景照“动起来”？

  1. 重启并进入 ComfyUI 客户端。
  2. 在（模板-视频）中下载 LTX-2.3: 图生视频 官方工作流。
  3. 载入后，将默认的 UNet / Checkpoint 主模型切换为你刚刚下载好的 Sulphur 2 模型。
  4. 上传你的本地图片，点击 Queue Prompt，静待魔法发生。

✍️ 玩法二：文生视频（Text2Vid）

  1. 下载专用的文生视频工作流：【点击下载】或【备用下载】
  2. 将下载好的 .json 工作流文件直接拖拽进 ComfyUI 界面。
  3. 在文本输入框（CLIP Text Encode）里用英文尽情放飞你的想象力（由于无审查，你可以尝试任何大胆的提示词）。
  4. 调整好步数（Steps）和分辨率，点击运行！

📝 极客总结

Sulphur 2 的出现再次证明了开源社区的生命力。虽然 8G 显存跑出来的视频可能在分辨率和帧率上有所妥协，但“能跑通本地无审查 AI 视频”本身就已经跨出了一大步。
