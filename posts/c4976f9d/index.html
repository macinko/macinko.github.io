openmediavault-backupжҸ’д»¶д»Ӣз»Қ

openmediavault-backup жҸ’д»¶жҳҜ OpenMediaVault (OMV) зі»з»ҹдёӯз”ЁдәҺеӨҮд»Ҫж•°жҚ®зҡ„ејәеӨ§е·Ҙе…·гҖӮе®ғеҸҜд»Ҙеё®еҠ©дҪ иҪ»жқҫең°еӨҮд»Ҫзі»з»ҹеҲҶеҢәгҖҒж•ҙдёӘзЈҒзӣҳжҲ–зү№е®ҡж–Үд»¶пјҢд»ҘзЎ®дҝқж•°жҚ®е®үе…ЁгҖӮд»ҘдёӢжҳҜ openmediavault-backup зҡ„дҪҝз”Ёж–№жі•пјҡ

1. е®үиЈ… openmediavault-backup жҸ’д»¶

  • йҰ–е…ҲпјҢзЎ®дҝқдҪ зҡ„ OMV зі»з»ҹе·ІиҝһжҺҘеҲ°дә’иҒ”зҪ‘гҖӮ
  • зҷ»еҪ• OMV зҡ„ Web з®ЎзҗҶз•ҢйқўгҖӮ
  • еҜјиҲӘеҲ°вҖңзі»з»ҹвҖқ>вҖңжҸ’д»¶вҖқгҖӮ
  • еңЁжҸ’д»¶еҲ—иЎЁдёӯжүҫеҲ° openmediavault-backup жҸ’д»¶пјҢе№¶зӮ№еҮ»вҖңе®үиЈ…вҖқгҖӮ
  • зӯүеҫ…жҸ’д»¶е®үиЈ…е®ҢжҲҗгҖӮ

2. й…ҚзҪ®еӨҮд»Ҫд»»еҠЎ

  • е®үиЈ…е®ҢжҲҗеҗҺпјҢеҜјиҲӘеҲ°вҖңжңҚеҠЎвҖқ>вҖңеӨҮд»ҪвҖқгҖӮ
  • зӮ№еҮ»вҖңж·»еҠ вҖқжҢүй’®еҲӣе»әдёҖдёӘж–°зҡ„еӨҮд»Ҫд»»еҠЎгҖӮ
  • еңЁвҖңеёёи§„вҖқйҖүйЎ№еҚЎдёӯпјҢи®ҫзҪ®д»ҘдёӢеҸӮж•°пјҡ
    • еҗҚз§°пјҡ дёәеӨҮд»Ҫд»»еҠЎжҢҮе®ҡдёҖдёӘжҸҸиҝ°жҖ§еҗҚз§°гҖӮ
    • еҗҜз”Ёпјҡ еӢҫйҖүжӯӨйҖүйЎ№д»ҘеҗҜз”ЁеӨҮд»Ҫд»»еҠЎгҖӮ
    • зӣ®ж Үпјҡ йҖүжӢ©еӨҮд»Ҫж•°жҚ®зҡ„еӯҳеӮЁдҪҚзҪ®гҖӮдҪ еҸҜд»ҘйҖүжӢ©жң¬ең°е…ұдә«ж–Үд»¶еӨ№гҖҒиҝңзЁӢ SSH жңҚеҠЎеҷЁжҲ– USB и®ҫеӨҮгҖӮ
    • и®ЎеҲ’пјҡ и®ҫзҪ®еӨҮд»Ҫд»»еҠЎзҡ„жү§иЎҢйў‘зҺҮпјҢдҫӢеҰӮжҜҸеӨ©гҖҒжҜҸе‘ЁжҲ–жҜҸжңҲгҖӮ
    • дҝқз•ҷпјҡ и®ҫзҪ®еӨҮд»Ҫж–Үд»¶зҡ„дҝқз•ҷжңҹйҷҗпјҢд»ҘйҒҝе…ҚеҚ з”ЁиҝҮеӨҡеӯҳеӮЁз©әй—ҙгҖӮ
  • еңЁвҖңйҖүйЎ№вҖқйҖүйЎ№еҚЎдёӯпјҢйҖүжӢ©иҰҒеӨҮд»Ҫзҡ„еҶ…е®№пјҡ
    • еҲҶеҢәпјҡ йҖүжӢ©иҰҒеӨҮд»Ҫзҡ„зі»з»ҹеҲҶеҢәгҖӮ
    • зЈҒзӣҳпјҡ йҖүжӢ©иҰҒеӨҮд»Ҫзҡ„ж•ҙдёӘзЈҒзӣҳгҖӮ
    • ж–Үд»¶/зӣ®еҪ•пјҡ йҖүжӢ©иҰҒеӨҮд»Ҫзҡ„зү№е®ҡж–Үд»¶жҲ–зӣ®еҪ•гҖӮ
  • ж №жҚ®дҪ зҡ„йңҖжұӮпјҢй…ҚзҪ®е…¶д»–й«ҳзә§йҖүйЎ№пјҢдҫӢеҰӮеҺӢзј©гҖҒеҠ еҜҶе’ҢжҺ’йҷӨйЎ№гҖӮ
  • зӮ№еҮ»вҖңдҝқеӯҳвҖқжҢүй’®гҖӮ

3. жү§иЎҢеӨҮд»Ҫд»»еҠЎ

  • еӨҮд»Ҫд»»еҠЎе°ҶжҢүз…§дҪ и®ҫзҪ®зҡ„и®ЎеҲ’иҮӘеҠЁжү§иЎҢгҖӮ
  • дҪ д№ҹеҸҜд»ҘжүӢеҠЁжү§иЎҢеӨҮд»Ҫд»»еҠЎпјҡ
    • еҜјиҲӘеҲ°вҖңжңҚеҠЎвҖқ>вҖңеӨҮд»ҪвҖқгҖӮ
    • йҖүжӢ©иҰҒжү§иЎҢзҡ„еӨҮд»Ҫд»»еҠЎгҖӮ
    • зӮ№еҮ»вҖңз«ӢеҚіиҝҗиЎҢвҖқжҢүй’®гҖӮ

4. жҒўеӨҚеӨҮд»Ҫж•°жҚ®

  • иҰҒжҒўеӨҚеӨҮд»Ҫж•°жҚ®пјҢдҪ йңҖиҰҒдҪҝз”Ёзӣёеә”зҡ„жҒўеӨҚе·Ҙе…·гҖӮ
  • еҜ№дәҺеҲҶеҢәжҲ–зЈҒзӣҳеӨҮд»ҪпјҢеҸҜд»ҘдҪҝз”Ё dd жҲ– fsarchiver зӯүе·Ҙе…·гҖӮ
  • еҜ№дәҺж–Үд»¶/зӣ®еҪ•еӨҮд»ҪпјҢеҸҜд»ҘзӣҙжҺҘеӨҚеҲ¶еӨҮд»Ҫж–Үд»¶еҲ°зӣ®ж ҮдҪҚзҪ®гҖӮ

5. еӨҮд»Ҫж–№жі•

openmediavault-backup жҸ’д»¶ж”ҜжҢҒеӨҡз§ҚеӨҮд»Ҫж–№жі•пјҡ

  • ddпјҡ з”ЁдәҺе…ӢйҡҶж•ҙдёӘеҲҶеҢәжҲ–зЈҒзӣҳзҡ„еҺҹе§Ӣжҳ еғҸгҖӮпјҲдјҡз”ҹжҲҗxxx.dd.zstпјү
  • fsarchiverпјҡ з”ЁдәҺеӨҮд»Ҫе’ҢжҒўеӨҚж–Үд»¶зі»з»ҹгҖӮпјҲдјҡз”ҹжҲҗxxx.fsaпјү
  • rsyncпјҡ з”ЁдәҺеҗҢжӯҘж–Үд»¶е’Ңзӣ®еҪ•гҖӮ
  • borgbackupпјҡ з”ЁдәҺеҲӣе»әйҮҚеӨҚж•°жҚ®еҲ йҷӨе’ҢеҠ еҜҶзҡ„еӨҮд»ҪгҖӮ

6. жіЁж„ҸдәӢйЎ№

  • еңЁй…ҚзҪ®еӨҮд»Ҫд»»еҠЎд№ӢеүҚпјҢиҜ·зЎ®дҝқдҪ жңүи¶іеӨҹзҡ„еӯҳеӮЁз©әй—ҙжқҘеӯҳеӮЁеӨҮд»Ҫж•°жҚ®гҖӮ
  • е®ҡжңҹжөӢиҜ•еӨҮд»ҪжҒўеӨҚиҝҮзЁӢпјҢд»ҘзЎ®дҝқеӨҮд»Ҫж•°жҚ®еҸҜз”ЁгҖӮ
  • иҖғиҷ‘дҪҝз”ЁиҝңзЁӢеӨҮд»Ҫзӣ®ж ҮпјҢд»ҘйҳІжӯўжң¬ең°еӯҳеӮЁи®ҫеӨҮж•…йҡңеҜјиҮҙж•°жҚ®дёўеӨұгҖӮ
  • ж №жҚ®дҪ зҡ„ж•°жҚ®йҮҚиҰҒжҖ§е’ҢеӯҳеӮЁйңҖжұӮпјҢйҖүжӢ©еҗҲйҖӮзҡ„еӨҮд»Ҫж–№жі•е’Ңи®ЎеҲ’гҖӮ

жҒўеӨҚеӨҮд»ҪпјҲfsarchiver,rsync,dd,borgbackupпјү

openmediavault-backup жҸ’д»¶зҡ„жҒўеӨҚиҝҮзЁӢеҸ–еҶідәҺдҪ дҪҝз”Ёзҡ„еӨҮд»Ҫж–№жі•гҖӮд»ҘдёӢжҳҜдёҖдәӣеёёи§ҒеӨҮд»Ҫж–№жі•зҡ„жҒўеӨҚжӯҘйӘӨпјҡ

1. дҪҝз”Ё fsarchiver жҒўеӨҚ

  • fsarchiverжҳҜдёҖз§Қз”ЁдәҺеӨҮд»Ҫе’ҢжҒўеӨҚж–Үд»¶зі»з»ҹзҡ„е·Ҙе…·гҖӮеҰӮжһңдҪ зҡ„еӨҮд»Ҫд»»еҠЎдҪҝз”ЁдәҶ fsarchiverпјҢдҪ еҸҜд»ҘжҢүз…§д»ҘдёӢжӯҘйӘӨжҒўеӨҚж•°жҚ®пјҡ

    • еҮҶеӨҮжҒўеӨҚзҺҜеўғпјҡ

      • дҪ йңҖиҰҒдёҖдёӘиғҪеӨҹи®ҝй—®еӨҮд»Ҫж–Үд»¶зҡ„ Linux зҺҜеўғгҖӮиҝҷеҸҜд»ҘжҳҜеҸҰдёҖдёӘ Linux зі»з»ҹпјҢд№ҹеҸҜд»ҘжҳҜ OMV зі»з»ҹжң¬иә«зҡ„ Live CD/USBгҖӮ
      • зЎ®дҝқдҪ е·Із»Ҹе®үиЈ…дәҶ fsarchiver е·Ҙе…·гҖӮ

    • жҒўеӨҚжӯҘйӘӨпјҡ

      • жү“ејҖз»Ҳз«ҜпјҢе№¶дҪҝз”Ё fsarchiver restfs е‘Ҫд»ӨжқҘжҒўеӨҚеӨҮд»ҪгҖӮ

      • е‘Ҫд»Өзҡ„еҹәжң¬ж јејҸеҰӮдёӢпјҡ

        • fsarchiver restfs [еӨҮд»Ҫж–Үд»¶и·Ҝеҫ„] id=0,dest=[зӣ®ж ҮеҲҶеҢә]

      • дҫӢеҰӮпјҡ

      • еҰӮжһңеӨҮд»Ҫж—¶и®ҫзҪ®дәҶеҜҶз ҒпјҢйңҖиҰҒж·»еҠ -c passwordеҸӮж•°гҖӮ

      • дҫӢеҰӮпјҡ

      • иҜ·еҠЎеҝ…е°ҸеҝғпјҢзЎ®дҝқзӣ®ж ҮеҲҶеҢәжӯЈзЎ®пјҢеҗҰеҲҷеҸҜиғҪдјҡиҰҶзӣ–й”ҷиҜҜзҡ„ж•°жҚ®гҖӮ

2. дҪҝз”Ё rsync жҒўеӨҚ

  • rsyncз”ЁдәҺеҗҢжӯҘж–Үд»¶е’Ңзӣ®еҪ•гҖӮеҰӮжһңдҪ зҡ„еӨҮд»Ҫд»»еҠЎдҪҝз”ЁдәҶ rsyncпјҢдҪ еҸҜд»ҘзӣҙжҺҘеӨҚеҲ¶еӨҮд»Ҫж–Үд»¶еҲ°зӣ®ж ҮдҪҚзҪ®гҖӮ
    • жҒўеӨҚжӯҘйӘӨпјҡ
      • жүҫеҲ°еӨҮд»Ҫж–Үд»¶жүҖеңЁзҡ„зӣ®еҪ•гҖӮ
      • дҪҝз”Ёж–Үд»¶з®ЎзҗҶеҷЁжҲ–е‘Ҫд»ӨиЎҢе·Ҙе…·пјҲеҰӮ cp жҲ– mvпјүе°ҶеӨҮд»Ҫж–Үд»¶еӨҚеҲ¶еҲ°зӣ®ж ҮдҪҚзҪ®гҖӮ
      • зЎ®дҝқж–Үд»¶жқғйҷҗе’ҢжүҖжңүжқғжӯЈзЎ®гҖӮ

3. дҪҝз”Ё dd жҒўеӨҚ

  • ddз”ЁдәҺе…ӢйҡҶж•ҙдёӘеҲҶеҢәжҲ–зЈҒзӣҳзҡ„еҺҹе§Ӣжҳ еғҸгҖӮдҪҝз”ЁddжҒўеӨҚдјҡе°Ҷзӣ®ж ҮеҲҶеҢәжҲ–зЈҒзӣҳе®Ңе…ЁиҰҶзӣ–дёәеӨҮд»Ҫж—¶зҡ„зҠ¶жҖҒпјҢжүҖд»ҘдёҖе®ҡиҰҒзЎ®и®Өзӣ®ж ҮеҲҶеҢәзҡ„жӯЈзЎ®жҖ§гҖӮ

    • жҒўеӨҚжӯҘйӘӨпјҡ

      • жү“ејҖз»Ҳз«ҜпјҢиҫ“е…Ҙд»ҘдёӢе‘Ҫд»Ө

      • dd if=[еӨҮд»Ҫж–Үд»¶и·Ҝеҫ„] of=[зӣ®ж ҮеҲҶеҢә]

      • дҫӢеҰӮпјҡ

      • иҜ·еҠЎеҝ…е°ҸеҝғпјҢdd е‘Ҫд»ӨйқһеёёејәеӨ§пјҢдҪҶд№ҹеҫҲеҚұйҷ©гҖӮеҰӮжһңзӣ®ж ҮеҲҶеҢәй”ҷиҜҜпјҢеҸҜиғҪдјҡеҜјиҮҙж•°жҚ®дёўеӨұгҖӮ

4. дҪҝз”Ё borgbackup жҒўеӨҚ

  • borgbackup жҳҜдёҖз§ҚйҮҚеӨҚж•°жҚ®еҲ йҷӨзҡ„еӨҮд»ҪзЁӢеәҸгҖӮ
    • жҒўеӨҚжӯҘйӘӨпјҡ

йҮҚиҰҒжҸҗзӨәпјҡ

  • еңЁжҒўеӨҚж•°жҚ®д№ӢеүҚпјҢиҜ·еҠЎеҝ…еӨҮд»ҪйҮҚиҰҒж•°жҚ®пјҢд»ҘйҳІжҒўеӨҚиҝҮзЁӢдёӯеҮәзҺ°ж„ҸеӨ–жғ…еҶөгҖӮ
  • жҒўеӨҚиҝҮзЁӢеҸҜиғҪдјҡиҰҶзӣ–зӣ®ж ҮдҪҚзҪ®зҡ„зҺ°жңүж•°жҚ®пјҢиҜ·еҠЎеҝ…е°Ҹеҝғж“ҚдҪңгҖӮ
  • е»әи®®еңЁжҒўеӨҚеүҚпјҢд»”з»Ҷйҳ…иҜ»OpenmediavaultеӨҮд»ҪжҸ’д»¶зҡ„ж–ҮжЎЈпјҢд»ҘзЎ®дҝқжӯЈзЎ®ж“ҚдҪңгҖӮ

еҪ“ dd еӨҮд»Ҫзҡ„ж–Үд»¶е…·жңү .zst еҗҺзјҖж—¶пјҢиҝҷж„Ҹе‘ізқҖиҜҘеӨҮд»Ҫжҳ еғҸе·ІдҪҝз”Ё zstd з®—жі•иҝӣиЎҢдәҶеҺӢзј©гҖӮиҰҒжҒўеӨҚжӯӨзұ»еӨҮд»ҪпјҢдҪ йңҖиҰҒе…Ҳи§ЈеҺӢзј©иҜҘжҳ еғҸпјҢз„¶еҗҺжүҚиғҪдҪҝз”Ё dd е°Ҷе…¶еҶҷе…Ҙзӣ®ж ҮеҲҶеҢәжҲ–зЈҒзӣҳгҖӮ

д»ҘдёӢжҳҜжҒўеӨҚжӯҘйӘӨпјҡ

1. еҮҶеӨҮжҒўеӨҚзҺҜеўғ

  • Linux зҺҜеўғпјҡ
    • дҪ йңҖиҰҒдёҖдёӘеҸҜд»Ҙи®ҝй—®еӨҮд»Ҫж–Үд»¶зҡ„ Linux зҺҜеўғгҖӮеҸҜд»ҘдҪҝз”ЁеҸҰдёҖдёӘ Linux зі»з»ҹпјҢд№ҹеҸҜд»ҘдҪҝз”Ё OMV зі»з»ҹзҡ„ Live CD/USBгҖӮ
    • зЎ®дҝқе·Із»Ҹе®үиЈ…дәҶ zstd е·Ҙе…·гҖӮеңЁеӨ§еӨҡж•° Linux еҸ‘иЎҢзүҲдёӯпјҢеҸҜд»ҘдҪҝз”ЁеҢ…з®ЎзҗҶеҷЁпјҲеҰӮ aptгҖҒyum жҲ– dnfпјүе®үиЈ…е®ғгҖӮ
  • еӨҮд»Ҫж–Үд»¶пјҡ
    • зЎ®дҝқзҹҘйҒ“ .zst еӨҮд»Ҫж–Үд»¶зҡ„е®Ңж•ҙи·Ҝеҫ„гҖӮ
  • зӣ®ж ҮеҲҶеҢә/зЈҒзӣҳпјҡ
    • жһҒе…¶йҮҚиҰҒпјҡ д»”з»ҶзЎ®и®ӨиҰҒжҒўеӨҚзҡ„зӣ®ж ҮеҲҶеҢәжҲ–зЈҒзӣҳгҖӮй”ҷиҜҜзҡ„зӣ®ж ҮжҢҮе®ҡдјҡеҜјиҮҙж•°жҚ®ж°ёд№…дёўеӨұгҖӮ
    • дҪҝз”Ё lsblk жҲ– fdisk -l е‘Ҫд»ӨжҹҘзңӢеҸҜз”Ёзҡ„зЈҒзӣҳе’ҢеҲҶеҢәгҖӮ

2. и§ЈеҺӢзј©еӨҮд»Ҫжҳ еғҸ

  • жү“ејҖз»Ҳз«Ҝпјҡ
    • еңЁ Linux зҺҜеўғдёӯжү“ејҖз»Ҳз«ҜгҖӮ
  • дҪҝз”Ё zstd и§ЈеҺӢзј©пјҡ

      • zstd -d [еӨҮд»Ҫж–Үд»¶и·Ҝеҫ„]

    • дҫӢеҰӮпјҡ

3. дҪҝз”Ё dd жҒўеӨҚи§ЈеҺӢзј©зҡ„жҳ еғҸ

  • жү§иЎҢ dd е‘Ҫд»Өпјҡ

    • дҫӢеҰӮпјҡ

    • йҮҚиҰҒиӯҰе‘Ҡпјҡ жӯӨе‘Ҫд»Өдјҡз«ӢеҚіејҖе§ӢиҰҶзӣ–зӣ®ж ҮеҲҶеҢәжҲ–зЈҒзӣҳгҖӮжү§иЎҢд№ӢеүҚпјҢиҜ·еҶҚж¬ЎзЎ®и®Өзӣ®ж ҮгҖӮ

  • зӯүеҫ…жҒўеӨҚе®ҢжҲҗпјҡ
    • жҒўеӨҚиҝҮзЁӢеҸҜиғҪйңҖиҰҒдёҖдәӣж—¶й—ҙпјҢе…·дҪ“еҸ–еҶідәҺжҳ еғҸж–Үд»¶зҡ„еӨ§е°Ҹе’Ңзӣ®ж Үи®ҫеӨҮзҡ„еҶҷе…ҘйҖҹеәҰгҖӮ
  • жё…зҗҶпјҡ

    • еңЁжү§иЎҢ dd е‘Ҫд»Өд№ӢеүҚпјҢиҜ·еҠЎеҝ…еӨҮд»Ҫд»»дҪ•йҮҚиҰҒж•°жҚ®гҖӮ
