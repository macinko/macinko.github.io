sudo apt-get install iptables

дјҡиҰҒжұӮжӮЁдҝқеӯҳеҪ“еүҚ/зҺ°жңүзҡ„ iptables и§„еҲҷпјҢеҲҶIP4е’ҢIP6пјҢеҸҜд»Ҙй»ҳи®Ө YES
