告别收费AList，欢迎开源OpenList：白嫖才是正义！

发表于2025-08-25|更新于2025-11-01|webalist
|阅读量:

各位资源管理爱好者、网盘达人、NAS狂热分子，听说了吗？
你最爱的AList，那个曾经免费开源、人人称赞的神器——它被卖了！

别揉眼，没看错，是“真被卖了”！曾经我们津津乐道的轻量级网盘列表工具，如今在资本的镰刀下，摇身一变，搞起了付费VIP。不是说收费不行，而是你我都清楚，“开源信仰+自由分享”才是白嫖党的底气来源！

不过别急着落泪，真正的猛人从不只靠一个工具过日子！
就在AList“转型”的那一刻，OpenList横空出世，霸气宣言：

我们不收费，我们不圈地，我们只是想做个纯粹的开源项目。

项目概述

🌟什么是 OpenList？

OpenList，一个基于 Go 语言打造的轻量级网盘文件列表程序，接口对标 AList，操作逻辑几乎一模一样，上手门槛极低。你可以用它连接阿里云盘、123云盘、蓝奏云、OneDrive、Google Drive 等等，各种网盘一网打尽，挂载预览、在线播放、权限控制应有尽有！

不管你是想建个个人网盘主页，还是公司内网文档管理，或者当成资源站盘点工具，OpenList都能轻松胜任，而且永久免费开源，不玩花的！

🧠为什么选择 OpenList，而不是继续忍 AList？

说白了，就一句话：理想不该被割韭菜换现金。

过去我们喜欢 AList，是因为它：

  • 好用
  • 免费
  • 社区活跃
  • 插件丰富

但当它转身拥抱盈利模式时，不少原始功能开始被VIP墙挡住，更新节奏也开始偏向“用户留存”和“商业变现”。

而 OpenList 告诉你：我们回归初心，只为用户服务。

它开源、它透明、它拒绝商业限制，甚至文档都写得清清楚楚，一看就很真诚。那种“不是在搞闭源副本赚钱，就是在搞新名字重新割你”的套路，OpenList一个都没沾边！

如果你使用群晖 NAS 或者飞牛 FnOS、1Panel、宝塔面板 等有图形界面的 Docker 管理工具，那么部署就特别简单了。如果你有自己的 VPS 服务器，或者在自己的电脑上，可以使用命令行来使用 Docker 命令行部署，其实也很简单：

方式一：Docker 命令行部署

方式二：Docker Compose 部署

环境变量参数的说明如下
名称默认值说明
PUID0运行身份 UID
PGID0运行身份 GID
UMASK022https://en.wikipedia.org/wiki/Umask
RUN_ARIA2是否同时运行 ARIA2，当镜像含有 aria2 环境时默认为 true，否则为 false
TZ默认为 UTC 时区，如想指定时区，可设置此变量，例如：Asia/Shanghai

设置管理员 (admin) 用户密码：

首次部署运行，在日志里面能看到“password is XXXX”的字样，这就是你的初始密码。你也可以通过命令行手动设置一个新的密码。

通过浏览器登录访问：

浏览器访问：http://你的IP:5244 使用用户 admin 和刚设置的密码登录

通过 WebDav 访问：

进入 OpenList 的设置，启用 WebDav 服务之后，我们就能在任意其他应用程序上通过 WebDAV 协议连接了。

URLhttp://主机IP地址:端口号/dav/
路径/dav
协议http
端口号与网页端一致
WebDAV用户名与网页端用户名一致
WebDAV密码与网页端密码一致

除了 Docker 部署之外，其实 Openlist 也可以原生安装，直接下载对应系统的二进制包。开发者也做了中文文档，连 Token 获取工具都给准备好了，感兴趣的可以去看看。

⚙️功能亮点一览：

  • 🌈 支持多种主流云盘：阿里云盘、OneDrive、123云盘、蓝奏云、Google Drive…
  • 🔐 支持文件夹加密、访问权限控制、API调用接口
  • 📂 UI界面简洁明了，支持主题换肤与自定义配置
  • 🚀 前端后端分离部署，性能优秀，轻量如风
  • 🔄 支持定时刷新、缓存策略，文件同步稳如老狗
  • 🎁 所有功能免费开源，绝无内购提示！

我们需要的不是“功能VIP”，而是“开源自尊”

OpenList 的诞生，就像给开源圈注入了一针强心剂，它不仅是 AList 的延续，更是对“自由使用、社区共建”的信仰守护。

在这个所有软件都恨不得变成SaaS订阅的时代，OpenList 像一个不肯低头的骑士，替我们守住了最后一块净土。

🎉最后：别再缅怀AList了，OpenList已经到来！

AList的“背叛”，或许只是程序员世界里再正常不过的结局。
但 OpenList 的诞生，是属于咱“自由白嫖联盟”的胜利！

🚀 快速部署教程、API文档、主题美化，统统都有！

别再让钱包遭殃，也别再用“阉割版工具”将就生活。
用 OpenList，重新找回那个自由、开源、无套路的数字世界！
