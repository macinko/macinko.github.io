Windows/Mac

以在 Windows 上配置 Docker 国内镜像为例，按照以下步骤进行：

  1. 打开 Docker 设置。可以在任务栏右下角找到 Docker 图标，右键单击该图标，然后选择“Settings”打开设置。
  2. 在设置界面中，选择“Docker Engine”选项卡，在该选项卡中找到“registry-mirrors”一栏。
