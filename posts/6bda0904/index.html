AI图像（增强、调整大小、移除、放大）Hugging Face项目

发表于2025-12-25|更新于2025-12-27|toolssoftwareaihuggingface
|阅读量:

image(enhance,resize,remove,upscale)

现实时间40秒， 增加GPU能力1s， 1小时可恢复90s。

Image Upscaling Playground🦆761Upscale images to higher resolution

Note 图片变大，有2倍和4倍选项，只变大，不改变清晰度

Image Resizer🏃9Resize an image to any desired dimensions

Note 图片尺寸变化，可大可小，界面不直观。需要手工输入大小(输入不合适图片会变形)，没有输入和输出的预览图，生成后只给一个下载链接。

InstantIR🖼86diffusion-based Image Restoration model

Note 图片修复，自动生成以后，用左下的free按钮，与右边中间的restoration按钮调整输出效果。没有高清效果。 可以自定义调节输出尺寸，宽和高默认是0，保持原尺寸不变。

Finegrain Image Enhancer🖼1.94kClarity AI Upscaler Reproduction

Note 模糊图片变清晰，参数较多，upscale factor可调大，生成图片尺寸不变，清晰度不如flux和tile。与原图一致性较好。

Flux.1-dev Upscaler🔎1.62kUpscale low-resolution images to high resolution

Note 模糊图片变清晰，随机种子关闭，默认参数即可，自动放大到2048大小。表情有轻微变化，用时约1分钟

Tile Upscaler V2😻153Enhance and upscale images with advanced settings

Note 模糊图片变清晰，有高级设置，图片尺寸，强化度等等

Background Removal🌘2.61kRemove background from images

Note 背景去除，速度快，没有参数选项，有左右效果比对预览。原图片有网址输入功能。需要png格式需要选file output

BRIA RMBG 2.0🐢858remove background from any image

Note 背景去除，速度快，有左右对比预览效果。右下有png格式下载，有图长按选择下载图片webp格式。

Flux.1-dev Controlnet Upscaler

Flux.1-dev Controlnet Upscaler 是一个基于Hugging Face平台的图像放大模型，它使用先进的深度学习技术来提高图像的分辨率，同时保持图像质量。该模型特别适合需要对图像进行无损放大的场景，如图像编辑、游戏开发、虚拟现实等。
