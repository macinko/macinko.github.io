发表于2024-08-23|更新于2024-11-27|oscentosubuntu
|阅读量:

1. 检查IP，开放端口

检测IP是否可用

如果连接不上机器，需在甲骨文管理后台刷IP直至获取到一个可用IP！

开放端口

安装网络工具
apt install net-tools

安装iptables
sudo apt-get install iptables

安装iptables-persistent
sudo apt-get install iptables-persistent

会要求您保存当前/现有的 iptables 规则，分IP4和IP6，可以默认 YES
一般情况下，ubuntu安装好的时候，iptables会被安装上，如果没有的话那就安装上吧

添加规则
iptables -I INPUT -p tcp --dport 80 -j ACCEPT

持久化保存规则

完成上述操作就可以永久打开我们需要的端口了

查看 与 关闭端口

通过 iptables -L -n --line-number 可以显示规则和相对应的编号
iptables -D INPUT 1 删除INPUT链编号为1的规则。

查看 防火墙状态

可以使用 sudo netstat -lnp 和 lsof -i:port 工具来识别打开端口后面的进程/二进制文件。

防火墙 开放 端口
sudo ufw allow 8080

防火墙 关闭 端口
sudo fuser -k 8080/tcp
